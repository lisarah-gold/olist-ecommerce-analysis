# Olist E-commerce Business, Customer & Delivery Experience Analytics Portfolio Project

## Project Purpose
This is a personal data analytics portfolio project based on the Olist public e-commerce dataset.
The project aims to demonstrate my skills in SQL, Python, data visualization, and basic business analysis.

## Data Source
Brazilian E-Commerce Public Dataset by Olist.

## Tech Stack
MySQL, SQL, Python, pandas, matplotlib, seaborn, Jupyter Notebook, Tableau Public, GitHub.

## Current Status
Phase 0: Project initialization and learning rules setup.

## Scope Note
This project uses historical public data for learning and portfolio purposes.
It does not represent Olist's current business performance or the Chinese e-commerce market.

## Current Results
No data has been downloaded, imported, or analysed yet.
